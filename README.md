# Apollo 11 Documentation

Educational documentation about NASA's Apollo 11 mission.

## Mission Overview
- **Launch Date:** July 16, 1969
- **Landing Date:** July 20, 1969 (Moon landing)
- **Crew Members:**
  - Neil Armstrong – Commander
  - Buzz Aldrin – Lunar Module Pilot
  - Michael Collins – Command Module Pilot
- **Moon Walk Duration:** Approximately 2.5 hours on the lunar surface.

### Key Highlights
- First human landing on the Moon.
- Historic quote: "That's one small step for a man, one giant leap for mankind."
- Mission duration: 8 days, 3 hours, 18 minutes.
- Returned safely to Earth, splashing down in the Pacific Ocean on July 24, 1969.

Further details and data files are provided in the `data/` directory.
